## NOTAS DE PRODUCAO

### Elementos visuais sugeridos:
- Animacoes simples mostrando cada etapa
- Screenshots reais do site (com dados de exemplo)
- Icones coloridos para os fundos (idoso = azul, crianca = verde)
- Barra de progresso mostrando os 4 passos
- Legenda/closed caption

### Estilo de animacao:
- Motion graphics leve e moderno
- Cores do IncentivaBR (azul #1E3A5F, laranja #F7941D, verde #00A859)
- Transicoes suaves entre cenas

### Opcoes de formato:
1. **Screencast comentado:** Gravacao de tela real do site
2. **Motion graphics:** Animacao 2D com personagem guia
3. **Misto:** Telas reais + graficos explicativos

### Musica de fundo:
- Instrumental leve, corporativo mas acolhedor
- Volume baixo para nao competir com a narracao
- Sugestao: estilo "explainer video" comum em tutoriais

### Versoes adicionais:
- **Versao 30 segundos:** Para redes sociais (apenas o conceito + CTA)
- **Versao 5 minutos:** Para quem quer mais detalhes
- **Versao vertical:** Para Stories/Reels
